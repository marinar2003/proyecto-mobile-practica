# Aplicación Móvil de Práctica 📱

Este repositorio contiene un proyecto desarrollado en **Android Studio** como parte de mi formación en la Tecnicatura Superior en Desarrollo de Software. El objetivo principal de este desarrollo es consolidar los pilares fundamentales del entorno nativo de Android, la gestión del ciclo de vida de las aplicaciones y el diseño de interfaces de usuario dinámicas.

## 🚀 Tecnologías y Herramientas utilizadas
* **IDE:** Android Studio (Ecosistema de desarrollo oficial).
* **Lenguaje:** Java 
* **UI Layouts:** XML (Uso de contenedores como `ConstraintLayout` y `LinearLayout` para garantizar vistas adaptables).
* **SDK:** Android SDK Tools.
* **Control de Versiones:** Git y GitHub.

## 🛠️ Conceptos e Implementaciones clave
* **Diseño Responsivo Mobile:** Creación de interfaces de usuario adaptativas para diferentes tamaños y orientaciones de pantalla.
* **Navegación e Interacción:** Manejo de componentes interactivos (Buttons, Inputs, TextViews) y comunicación básica entre flujos de la aplicación.
* **Ciclo de Vida de Android:** Comprensión práctica de los estados de una Activity (`onCreate`, `onStart`, `onResume`, etc.) para una correcta gestión de memoria y persistencia de interfaz.
